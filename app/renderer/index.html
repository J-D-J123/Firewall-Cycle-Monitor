Proxy

Route system traffic through the monitor
—

When on, Windows sends browser & app traffic through the local proxy so it can be inspected, modified and blocked.

Startup

Launch when I sign in to Windows
—

Starts automatically at sign-in, minimized to the tray. It asks for administrator access each time (needed for real per-app firewall blocking). While this is on, all network traffic is blocked at startup until the proxy is active, so nothing leaks before protection is up. Turning the switch on/off doesn't need admin.

Guard new apps

Block apps I haven't approved
—

When a program you've never allowed before tries to use the network, its traffic is blocked and a prompt appears in the corner so you can allow or block it. Core Windows system processes are exempt.
